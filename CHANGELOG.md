# 更新日志

## v0.7.1

修复角色删除、网关重启等多个问题，并改善更新与定时任务体验。

### 新功能

- 支持运行时模型热补丁，无需重启应用即可切换模型

### 问题修复

- 删除角色不再删除聊天历史
- 修复网关重启时数据突变问题
- 支持本地 OpenClaw 网关验证
- 优化定时任务控制台输出
- 改善 Windows 网关启动速度
- 优雅处理已删除的技能详情
- 支持 BOM 格式的技能 frontmatter
- 明确定时任务状态显示
- 同步定时任务的网关设备标识
- 修复从微信小助手角色切换到主角色后无法聊天的问题
- 防止同一微信账号被重复绑定

## v0.6.6

引入 LibreOffice 聊天预览、隐藏开发者控制台，并修复多个稳定性问题。

### 新功能

- 支持预览 docx、xlsx、pptx 等文件格式的 LibreOffice 聊天预览
- 默认隐藏 OpenClaw 控制台入口，仅开发者模式可见

### 问题修复

- 修复网关重启时无提示的问题
- 修复定时任务微信投递就绪判断
- 修复 Office 预览时文件下载问题
- 修复 API Key 和 URI 错误校验
- 修复 Agent 删除交互体验
- 修复 CRLF 技能元数据解析问题
