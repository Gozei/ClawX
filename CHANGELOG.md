# 更新日志

## v0.7.1

**一句话总结：** 修复角色删除、网关重启等多个问题，并改善更新与定时任务体验。

### 新功能

- 更新对话框优化 — 支持运行时模型热补丁，无需重启应用即可切换模型

### 问题修复

- 删除角色不再删除聊天历史 — 避免误删用户对话记录
- 修复网关重启时数据突变 — 防止并发写入导致状态异常
- 支持本地 OpenClaw 网关验证 — 开发者可离线调试网关连接
- 优化定时任务控制台输出 — 状态展示更清晰
- 改善 Windows 网关启动速度 — 减少启动等待时间
- 优雅处理已删除的技能详情 — 避免访问已失效技能时崩溃
- 支持 BOM 格式的技能 frontmatter — 兼容带 BOM 编码的 YAML 文件
- 明确定时任务状态显示 — 解决状态模糊导致用户困惑的问题
- 同步定时任务的网关设备标识 — 修复跨设备定时任务执行异常
- 修复微信小助手切换后无法聊天 — 从副角色切回主角色后通信恢复正常
- 防止同一微信账号重复绑定 — 增强绑定校验逻辑

## v0.6.6

**一句话总结：** 引入 LibreOffice 聊天预览、隐藏开发者控制台，并修复多个稳定性问题。

### 新功能

- LibreOffice 聊天预览 — 支持预览 docx、xlsx、pptx 等文件格式，无需外部依赖
- 隐藏 OpenClaw 控制台 — 默认隐藏控制台入口，仅开发者模式可见

### 问题修复

- 修复网关重启提示缺失 — 网关中断后不再无提示，提示更明确
- 修复定时任务微信投递就绪判断 — 定时任务正确等待通道就绪再投递
- 修复 Office 预览时文件下载问题 — 处理下载地址和参数编码问题
- 修复 API Key 和 URI 错误校验 — 允许携带格式问题的配置通过基本校验
- 修复 Agent 删除交互 — 删除流程增加确认提示
- 修复 CRLF 技能元数据解析 — 兼容 Windows 换行符的 YAML 文件
