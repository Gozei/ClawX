# 中文PPT深度设计指南

> 本指南不是一份"操作手册"，而是一套设计思维的传递。目标是让AI在生成每一页PPT时，都能像真正理解设计的人一样思考——知道为什么这样做，而不仅仅是知道怎么做。

---

## 一、设计哲学：六大底层原则

### 原则1：色彩即语义，不是装饰

颜色不是"好看就行"。每份PPT的主色调必须与内容主题产生语义关联：

- **蓝色 = 信任与专业**：工作汇报、年终总结、商务提案
- **红色 = 权威与使命**：党政宣讲、团课、党建活动
- **绿色 = 成长与生态**：农业项目、环保主题、可持续发展
- **深灰/黑 = 极致与学术**：毕业论文答辩、学术文献汇报、极简风格
- **多彩 = 活力与体验**：旅游指南、产品发布、创意提案

**反面教材**：用默认蓝色做党政PPT，用红色做学术答辩。

### 原则2：层次感来自透明叠加，而非纯色堆砌

10份PPT中，**没有一份**使用100%不透明的大面积纯色块。真正的层次感来自：

- **半透明渐变叠加**：一个30-60%透明度的渐变色块叠加在图片或底色上，创造"呼吸感"
- **透明渐变**：从`rgba(color, 0.3)`渐变到`rgba(color, 0)`，让色块边缘自然消融
- **多层叠加**：一个半透明色块+另一个不同角度的半透明色块=丰富的视觉层次

**具体参数**：
- 装饰色块透明度：30-60%（太高=沉闷，太低=看不见）
- 渐变方向：通常从实色渐变到透明（左→右或上→下）
- 叠加层数：2-3层即可，不要超过4层

### 原则3：阴影创造空间感，但必须克制

几乎所有PPT都使用外阴影（outerShdw），但参数差异极大：

| 场景 | blurRad | dist | dir(方向) | 效果 |
|------|---------|------|-----------|------|
| 精致商务 | 50800(4pt) | 25400(2pt) | 2700000(左上) | 轻微浮起感 |
| 自然柔和 | 381000(30pt) | 38100(3pt) | 5400000(正下) | 柔和投影 |
| 深度空间 | 1270000(100pt) | 1092200(85pt) | 8100000(右下) | 强烈立体感 |
| 极简学术 | 254000(20pt) | 190500(15pt) | 13500000(正下) | 干净利落投影 |

**原则**：同一PPT内阴影参数必须统一。不要一页用轻微阴影、下一页用深度阴影。

### 原则4：节奏感来自"呼吸点"

每2-4页内容后，必须插入一个章节过渡页。这不是"浪费页数"，而是：

- 给观众**视觉休息**（密集信息后的空白）
- **重新锚定注意力**（切换话题前让观众"回神"）
- **强化叙事结构**（让观众知道自己在哪里）

**最佳节奏**：`封面 → 目录 → [过渡页 → 2-3页内容] × N → 结语`

### 原则5：一致性 > 多样性

一份PPT看起来"专业"还是"杂乱"，取决于一致性：

- **LOGO位置**：每页固定在相同位置（推荐右上角或左上角）
- **顶部/底部色带**：贯穿全PPT，颜色和高度不变
- **标题位置和字号**：所有内容页标题在相同Y坐标
- **章节编号格式**：统一用"PART 01"或"第壹章"或"01"，不要混用
- **页码位置**：统一在底部色带内或底部右侧

### 原则6：封面与结语必须呼应

结语/致谢页不是"随便放个谢谢"。它必须与封面产生**视觉呼应**：
- 使用相同的背景图或装饰元素
- 使用相同的排版（主标题+副标题+署名）
- 主标题换成"感谢观看"/"THANK YOU"等
- 给观众"圆满结束"的心理暗示

## 二、色彩体系：具体到hex值

### 场景一：工作汇报（通用商务）

**主色调**：深蓝系，传递专业、可靠、冷静

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#005EA4` | 顶部色带、封面背景、章节过渡页 |
| 浅主色 | `#007AD6` | 渐变终点、按钮、链接 |
| 亮色 | `#097CFA` | 强调数据、图标背景 |
| 极浅色 | `#62ACFC` → `#93C6FD` | 装饰色块（透明度20-40%） |
| 辅助浅色 | `#F9FCFD` | 内容区背景 |
| 强调色 | `#D53D2A` | 关键数据标注、警示信息 |
| 文字深色 | `#1A2029` | 标题、正文 |
| 文字浅色 | `#E8E8E8` | 深色背景上的文字 |

**渐变用法**：`#005EA4` → `#007AD6`（左到右），叠加透明度3-7%的白色渐变创造光泽感

**阴影参数**：blurRad=127000(10pt), dist=127000(10pt), dir=5400000(正下) — 柔和商务感

### 场景二：年终总结

**主色调**：同工作汇报蓝色系，但加入温暖元素

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#004BA0` | 大面积色块 |
| 中蓝 | `#4377A7` | 卡片边框、分隔线 |
| 浅蓝 | `#56A8BD` | 装饰点缀 |
| 点缀绿 | `#6C844A` | 正面数据标注 |
| 点缀红 | `#7C0A22` | 警示、未达标数据 |
| 背景色 | `#F8F8F8` → `#FBFBFB` | 内容区 |

**渐变用法**：半透明白色(`#FFFFFF`, 透明度20%)渐变到透明，叠加在蓝色块上

**阴影参数**：blurRad=381000(30pt), dist=190500(15pt), dir=3000000(左上偏下) — 精致微妙

### 场景三：党政/团课

**主色调**：中国红系，传递权威、使命感、热情

| 角色 | 色值 | 用途 |
|------|------|------|
| 深红 | `#A00904` | 渐变终点、暗部 |
| 主红 | `#C00000` | 主色调、色带 |
| 亮红 | `#D30013` | 强调、按钮 |
| 亮金 | `#E10802` | 渐变起点、高光 |
| 辅助色 | `#262626` | 深色文字、边框 |
| 浅底色 | `#FEFEFE` | 内容区背景 |

**渐变用法**：`#E10802` → `#A00904`（红色系内渐变），叠加透明度15-46%的渐变色块

**阴影参数**：blurRad=50800(4pt), dist=38100(3pt), dir=2700000(左上) — 紧凑精致

### 场景四：学术答辩

**主色调**：极简黑白或深灰系，传递严谨、客观

| 角色 | 色值 | 用途 |
|------|------|------|
| 极深灰 | `#231915` | 封面背景 |
| 深灰 | `#313132` → `#3B3B3D` | 标题文字 |
| 中灰 | `#545456` | 正文文字 |
| 浅灰 | `#C8C9C9` → `#E8E8E8` | 分隔线、装饰 |
| 极浅灰 | `#F2F2F2` → `#FAFAFA` | 内容区背景 |

**渐变用法**：透明渐变（从30%透明度渐变到0），极少使用有色渐变

**阴影参数**：blurRad=254000(20pt), dist=190500(15pt), dir=13500000(正下) — 干净利落
**透明度用法**：15%轻覆盖、50%中等覆盖、75%强覆盖

### 场景五：农业/环保

**主色调**：多色系但以绿色为主，传递自然、成长

| 角色 | 色值 | 用途 |
|------|------|------|
| 深绿 | `#3E6000` | 深色装饰 |
| 森林绿 | `#40693A` → `#4F8A4D` | 主色调 |
| 亮绿 | `#438F24` → `#AEC056` | 强调、图标 |
| 天蓝 | `#089FD0` → `#43C5F4` | 辅助色、数据 |
| 暖红 | `#A5190D` → `#CF4237` | 对比强调 |
| 金色 | `#E4BC57` | 数据高亮 |
| 深底色 | `#1A2029` → `#1A2D4F` | 深色背景页 |

**渐变用法**：绿色渐变(`#3E6000` → `#AEC056`)，叠加蓝色透明色块，透明度7-15%（极微妙）

**阴影参数**：blurRad=25400(2pt), dist=25400(2pt), dir=2700000(左上) — 极轻微

### 场景六：旅游/生活方式

**主色调**：极简，几乎完全靠图片讲故事

| 角色 | 色值 | 用途 |
|------|------|------|
| 纯黑 | `#000000` | 文字 |
| 白色 | `#FFFFFF` | 覆盖层文字底色（半透明） |

**设计要点**：
- 图片占页面80%+面积
- 文字通过半透明白色/黑色底色衬托在图片上
- 阴影极多(blurRad=127000-139700, dist=38100, 多方向)，营造"杂志拼贴"感
- 透明度14-59%，让图片和文字自然融合

## 三、叙事结构与内容架构

### 3.1 通用叙事弧线

每份PPT都遵循"起承转合"的叙事弧线：

```
起（开场）    → 封面：建立第一印象，定调
承（框架）    → 目录：让观众知道"路线图"
转（展开）    → 各章节：层层递进，每章2-4页
  ├ 过渡页    → 节奏休息 + 话题切换
  └ 内容页    → 核心信息传递
合（收束）    → 结语：呼应开场 + 情感连接
```

### 3.2 工作汇报类标准结构

**适用**：月度/季度/年度工作汇报、述职报告、年终总结

```
封面
  ├ 主标题：工作总结汇报 / 年终工作总结 / 述职报告
  ├ 英文副标题：WORK REPORT / ANNUAL SUMMARY
  ├ 汇报人 + 日期 + 单位/部门
  └ LOGO

目录（3-4项）
  ├ 01 工作概况 / 阶段工作概述
  ├ 02 工作成果 / 重点成果展示
  ├ 03 问题分析 / 不足与改进
  └ 04 未来计划 / 工作规划

PART 01 工作概况（2-3页）
  ├ 过渡页：PART 01 + 标题 + 英文副标题
  ├ 内容页1：工作概述（文字+关键数据）
  ├ 内容页2：完成情况/数据图表
  └ [可选] 内容页3：调整计划/重点项目

PART 02 工作成果（2-3页）
  ├ 过渡页
  ├ 内容页1：成果概述（左文右图或三栏卡片）
  ├ 内容页2：数据展示（图表+大号数字）
  └ [可选] 内容页3：重点项目详情

PART 03 问题分析（2-3页）
  ├ 过渡页
  ├ 内容页1：问题梳理（列表或卡片）
  ├ 内容页2：原因分析（左右对比或SWOT）
  └ [可选] 内容页3：改进措施

PART 04 未来计划（2-3页）
  ├ 过渡页
  ├ 内容页1：核心目标
  ├ 内容页2：具体计划/时间线
  └ [可选] 内容页3：风险预案

结语（1页）
  ├ 与封面呼应的视觉设计
  ├ "谢谢观看" / "感谢聆听"
  ├ 汇报人 + 联系方式
  └ LOGO
```

### 3.3 学术答辩类标准结构

**适用**：本科/硕士/博士毕业答辩、开题报告、学术汇报

```
封面
  ├ 论文题目（居中，大号）
  ├ 英文标题
  ├ 答辩人 + 指导老师
  └ 学校/学院 LOGO

目录（4-5项，编号式）
  ├ 一、选题背景及意义
  ├ 二、研究思路及方法
  ├ 三、研究成果展示及应用
  └ 四、研究结论与致谢

PART 01 选题背景及意义（2页）
  ├ 过渡页 + 副标题"对研究课题的背景和重要性的阐述"
  ├ 内容页1：课题背景（行业背景、问题提出）
  └ 内容页2：课题意义/研究目标

PART 02 研究思路及方法（2-3页）
  ├ 过渡页 + 副标题"研究课题的具体实施策略"
  ├ 内容页1：研究思路/理论框架
  ├ 内容页2：研究方法/工具
  └ [可选] 内容页3：技术路线图

PART 03 研究成果（2-3页）
  ├ 过渡页 + 副标题"研究过程中取得的主要成果"
  ├ 内容页1：研究成果（图文+数据）
  ├ 内容页2：成果应用/案例
  └ [可选] 内容页3：研究局限

PART 04 结论与致谢（1-2页）
  ├ 过渡页
  ├ 内容页1：研究结论
  └ 结语页："感谢聆听 敬请指正" + 答辩人
```

### 3.4 党政/团课类标准结构

**适用**：党课、团课、政治学习、组织生活

```
封面
  ├ 主标题（如"党政红色团课PPT模板"）
  ├ 副标题
  ├ 汇报人 + 时间
  └ [可选] 党徽/团徽

目录（4项，中式排版）
  ├ 党史的光辉历程回顾
  ├ 传承伟大的革命精神
  ├ 新时代党员先锋模范
  └ 共青团员的使命担当

各章节（每章2-4页内容）
  ├ 过渡页：PART ONE/TWO/THREE/FOUR + 标题
  ├ 内容页：配图 + 要点文字
  └ [可选] 内容页：数据/案例

结语（1页）
  ├ "非常感谢您的观看"
  └ 与封面呼应
```

### 3.5 年终总结的高级结构（P08范例）

P08展示了最高级的结构设计——每章都有一个**叙事角度标签**：

```
{核心数据 + 关键突破} → 年度工作成果回顾    ← 用数据说话
{客观复盘 + 深度拆解} → 问题反思与原因分析    ← 深度反思
{目标拆解 + 落地路径} → 下年度工作计划        ← 可执行计划
{情感共鸣 + 价值传递} → 总结与致谢            ← 情感收束
```

这种"花括号标签"的设计让观众一眼就知道每个部分的**叙事角度**，极大提升了PPT的专业感。**建议在所有工作汇报类PPT中采用这种结构标签**。

## 四、文案写作风格指南

### 4.1 封面文案

**工作汇报类**：简洁庄重
```
主标题：工作总结汇报（或年度工作汇报、述职报告）
英文副标题：WORK REPORT / ANNUAL WORK REPORT / PERFORMANCE REVIEW
署名行：汇报人：XXX  |  日期：20XX.X  |  部门：XXX
```

**学术答辩类**：规范正式
```
主标题：论文题目（居中，30-40字以内）
英文标题：UNDERGRADUATE THESIS PROPOSAL DEFENSE
署名行：答辩人：XXX  |  指导老师：XXX
```

**党政宣讲类**：权威大气
```
主标题：党政红色团课PPT模板（或具体主题名称）
副标题：请在此处添加演示文稿副标题
署名行：汇报人：XXX  |  时间：20XX.X
```

**旅游/创意类**：文艺感性
```
主标题：出发遇见未来
感性副标题：说走就走    是人生最华简的奢侈
英文点缀：Set off and meet the future
```

### 4.2 章节标题的命名逻辑

**四字词组**（最常用，简洁有力）：
- 工作概况、完成情况、调整计划、工作成果、优点缺点、未来展望
- 选题背景、研究思路、研究成果、结论致谢

**"名词+动词"结构**（正式感更强）：
- 乡村振兴战略背景、技术创新赋能、模式创新赋能
- 农业项目实施保障、资源整合与配置、政策建议与措施

**"行动+目标"结构**（年度总结专用）：
- {核心数据+关键突破}年度工作成果回顾
- {客观复盘+深度拆解}问题反思与原因分析

**品牌口号式**（政务/党建专用）：
- 【乡村振兴农为本】
- 传承伟大的革命精神
- 新时代党员先锋模范

### 4.3 内容描述的写作风格

**工作汇报类**：概括性语言，突出结果
- ✅ "全年签约额2100万元，开拓12家新行业客户（含3家头部企业）"
- ✅ "超额完成年度目标，客户满意度92分，首次解决率72%"
- ❌ 不要写流水账式的日常工作描述

**学术类**：正式严谨，带论证逻辑
- ✅ "根据事物运动规律和行业发展趋势，某某问题是行业发展的趋势"
- ✅ "在实际运作中，常常遇到某某问题，且有愈发严重的态势"
- ❌ 不要用口语化表达

**政务类**：政策性语言，体现权威
- ✅ "旨在提高农业生产效率、增加农民收入、促进农村经济发展"
- ✅ "乡村振兴战略为农业项目提供了政策支持、市场机遇和资源保障"
- ❌ 不要用主观评价

**旅游/创意类**：生动具体，有画面感
- ✅ "黄山年云雾天数达250余天，云海翻腾，奇松怪石掩映其间，被誉为天下第一奇山"
- ✅ "小众景点如束河古镇，保留90%以上原住民，年游客量仅500万，体验地道"
- ❌ 不要用笼统的"很美""值得去"

### 4.4 结语文案

| 场景 | 推荐文案 | 英文搭配 |
|------|---------|---------|
| 工作汇报 | 感谢聆听 | THANK YOU |

## 六、各场景设计策略速查

### 工作汇报/年终总结
- **色彩**：蓝色系(#005EA4-#93C6FD)+红色强调(#D53D2A)
- **结构**：概况→成果→问题→计划，每章加"叙事角度标签"
- **文案**：数据驱动，突出具体数字和百分比
- **布局**：左文右图+三栏卡片+数据图表交替
- **节奏**：每3页内容插入1个过渡页

### 学术答辩/开题
- **色彩**：极简灰白(#231915-#FAFAFA)或纯黑+白
- **结构**：背景→方法→成果→结论，层级编号(1.1/1.2/2.1)
- **文案**：正式严谨，每段有论证逻辑
- **布局**：简洁大方，避免花哨装饰，重点在内容
- **节奏**：每2页内容插入1个过渡页（学术类信息密度低）

### 党政/团课
- **色彩**：中国红(#C00000-#D30013)+金色点缀(#E4BC57)+深灰(#262626)
- **结构**：党史回顾→革命精神→先锋模范→使命担当
- **文案**：政策性语言+品牌口号
- **布局**：大图+文字叠加，红色渐变背景
- **节奏**：每3-4页内容插入1个过渡页（党政类内容丰富）

### 商业/产品发布
- **色彩**：深色系(#24262C-#475474)+亮色强调(#D5251A)
- **结构**：市场背景→产品方案→竞争优势→行动计划
- **文案**：简洁有力，数据说话，避免冗长描述
- **布局**：大图+极简文字，营造高端感
- **节奏**：每2页内容插入1个过渡页

### 旅游/创意展示
- **色彩**：极简（几乎全靠图片），文字黑/白
- **结构**：景点介绍→体验攻略→美食地图→推荐清单
- **文案**：生动具体有画面感，用数据增强可信度
- **布局**：全宽图片+文字叠加，多阴影营造杂志感
- **节奏**：每2页内容插入1个"诗意过渡页"

## 七、信息密度与复杂度控制

### 7.1 每页形状数量是"专业感"的隐藏指标

从10份PPT的统计数据中，发现了一个清晰的规律：**每页形状数量直接决定了PPT的视觉复杂度**。

| 场景 | 平均形状数/页 | 视觉感受 | 设计策略 |
|------|-------------|---------|----------|
| 极简学术(P05) | 2-3 | 极简、内容为王 | 每页仅标题+1个主体元素 |
| 通用工作汇报(P01) | 3-5 | 简洁大气 | 每页标题+1-2个内容模块 |
| 红金商务(P02) | 8-10 | 精致丰富 | 每页标题+2-3个图文模块 |
| 深灰学术(P06) | 15-18 | 信息密集 | 多栏布局+数据卡片 |
| 年终总结(P08) | 17-18 | 数据驱动 | 图表+数据标签+说明 |
| 党政团课(P09) | 16-17 | 内容丰富 | 图片+要点+装饰 |
| 简约工作总结(P03) | 18 | 复杂但有序 | 多种布局交替 |
| 农业项目(P04) | 20 | 图文并重 | 大量图片+数据 |
| 旅游指南(P10) | 20 | 图片驱动 | 多图拼贴+少量文字 |

**规律**：学术类极简(2-3)，商务类中等(8-18)，图文并茂型密集(18-20)。

**关键原则**：同一PPT内，各页的形状数量差异不应超过2倍。如果一页有5个形状，下一页不应突然有30个。

### 7.2 图片使用策略

| 场景 | 图片数量/页 | 平均图片尺寸 | 图片占比 | 策略 |
|------|-----------|------------|---------|------|
| 学术答辩(P05) | 0 | — | 0% | 纯文字+图表，不用装饰图片 |
| 工作总结(P07) | 0.05 | 0.7×0.7in | 极低 | 仅用极小图标点缀 |
| 深灰学术(P06) | 0 | — | 0% | 灰度渐变+阴影代替图片 |
| 通用商务(P02) | 0.7 | 10.0×2.6in | 中等 | 宽幅横图条作为装饰带 |
| 农业项目(P04) | 1.3 | 7.2×3.9in | 中高 | 左文右图、图文混排 |
| 年终总结(P08) | 1.0 | 2.9×2.1in | 中等 | 小图+数据图表配合 |
| 党政团课(P09) | 1.3 | 9.8×3.0in | 中高 | 宽幅配图+红色叠加 |
| 简约总结(P03) | 1.1 | 4.3×5.0in | 中等 | 竖版图配文字说明 |
| 旅游指南(P10) | 4.4 | 2.0×4.0in | 极高 | 多图拼贴，图片=主角 |

**图片尺寸规律**：
- **商务/政务**：偏好宽幅横图(9-10英寸宽×2-3英寸高)，作为页面装饰带
- **农业/总结**：中等尺寸(4-7英寸宽×4英寸高)，与文字并排
- **旅游**：小图多图(2英寸宽×4英寸高)，杂志拼贴风格
- **学术**：基本不用装饰图片

### 7.3 图片边框规则

| 场景 | 图片有无边框 | 边框宽度 | 边框颜色 | 风格 |
|------|------------|---------|---------|------|
| 学术(P05/P06) | 无边框 | — | — | 极简 |
| 年终总结(P08) | 无边框 | — | — | 图片与背景自然融合 |
| 旅游(P10) | 有边框(21%) | 1.0-1.8pt | #000000 | 虚线(sysDash)，杂志感 |
| 党政(P09) | 有边框(21%) | 0.5-6pt | #D8261E/#C00000 | 红色实线/虚线 |
| 农业(P04) | 有边框(22%) | 0.5-3pt | #4F8A4D/#E4BC57 | 绿色/金色，主题色 |
| 商务(P02) | 有边框(18%) | 1.5-4pt | #B3110D | 红色实线(solid) |

- 学术/极简风格 = 不加边框

- 创意/拼贴风格 = 加虚线边框，营造手账/杂志感

| 场景 | 圆角形状占比 | 圆角程度 | 风格 |

### 7.5 边框线条样式

| 场景 | 线条样式 | 线条宽度范围 | 颜色 |

### 7.6 形状组合(GROUP)策略

| 场景 | 平均组合数/页 | 平均子元素数 | 最大子元素 | 策略 |

### 7.8 视觉重量分布（左重 vs 右重 vs 平衡）

从10份PPT的形状面积分布分析得出：

| 场景 | 主导分布 | 说明 |
|------|---------|------|
| 极简学术(P05) | 完全居中 | 视觉元素集中在页面中央 |
| 工作总结(P07) | 平衡+右重过渡 | 内容页平衡，过渡页右重 |
| 年终总结(P08) | 平衡+右重 | 多数页面左右平衡，过渡页偏右 |
| 旅游指南(P10) | 平衡+交替 | 左重和右重交替出现，避免单调 |
| 农业项目(P04) | 左重+平衡交替 | 内容页左重（文字优先），数据页平衡 |
| 简约总结(P03) | 左重为主 | 文字区在左，视觉区在右 |
| 深灰学术(P06) | 左重+平衡 | 左侧文字为主，图表页平衡 |
| 党政团课(P09) | 左重为主 | 装饰元素在左，内容在右 |

**规律**：
- **中文阅读从左到右** → 文字内容自然偏左（占55-60%页面宽度）
- **图片/图表** → 偏右（占40-45%页面宽度）
- **过渡页** → 可以居中或右偏，打破内容页的左重模式
- **绝对不要**：所有页面都左重 → 视觉疲劳。交替使用平衡和左右偏移

### 8.2 布局组合的黄金法则

每份PPT都不是只用一种布局，而是**有策略地组合多种布局类型**：

**工作汇报类**（P03/P07/P08模式）：
```
过渡页(MINIMAL) → 平衡布局(BALANCED) → 数据展示(DATA) → 平衡(BALANCED)
→ 过渡页(MINIMAL) → 文字密集(TEXT) → 复杂组合(COMPLEX) → 平衡(BALANCED)
```
比例约为：过渡页25% + 平衡25% + 数据/文字20% + 复杂30%

**政务/党建类**（P09模式）：
```
过渡页(MINIMAL) → 平衡(BALANCED) → 复杂组合(COMPLEX) → 图片主导(IMAGE)
→ 过渡页(MINIMAL) → 平衡(BALANCED) → 数据展示(DATA) → 复杂组合(COMPLEX)
```
比例约为：过渡页35% + 平衡20% + 复杂30% + 图片/数据15%

**学术类**（P05/P06模式）：
```
过渡页(MINIMAL) → 平衡(BALANCED) → 文字密集(TEXT) → 平衡(BALANCED)
→ 过渡页(MINIMAL) → 文字密集(TEXT) → 复杂组合(COMPLEX) → 平衡(BALANCED)
```
比例约为：过渡页35% + 平衡25% + 文字密集25% + 复杂15%

**旅游/创意类**（P10模式）：
```
过渡页(MINIMAL) → 图片主导(IMAGE) → 平衡(BALANCED) → 图片主导(IMAGE)
→ 过渡页(MINIMAL) → 平衡(BALANCED) → 复杂组合(COMPLEX) → 图片主导(IMAGE)
```
比例约为：过渡页30% + 平衡40% + 图片主导20% + 复杂10%

**原则**：
- 过渡页(MINIMAL)占比固定在25-35%，是节奏的基石
- 永远不要连续出现2页相同布局类型
- 每个章节内部至少有2种不同布局
- 数据展示页和文字密集页不要连续超过2页
