# LHC统计技能

## 描述

彩票投注记录解析统计工具，支持多种格式的六合彩投注记录解析和统计。

## 功能

- 解析聊天记录格式的投注记录
- 支持多种玩法：三中三、二中二、死活组、特串
- 自动合并相同组合，统计组数和金额
- 支持混合玩法（一行多种玩法）
- 支持分行记录（号码和金额分两行）

## 使用方法

### 通过 LHC Agent

```
直接发送图片或文字记录给 LHC Agent
```

### 命令行

```bash
python scripts/lhc_parse.py <投注记录文件>
python scripts/lhc_parse.py "<投注记录文字>"
```

## 支持的格式

### 单组三中三
```
24.03.37三中三300
```

### 复式三中三
```
10-15-17-20-25-29-44 各50
```

### 混合玩法
```
09-49-36-37-27-11-06 复三中三名2,二中二各5,特串各2
```

### 分行记录
```
3-15-27-39-8-20-32-44
复试三中三各5
```

### 死活组
```
新20--42--37--28 死活组各40
```

## 输出格式

```
数字1 数字2 数字3 共计组数 共计金额

示例：
03 24 37 1组 300元
10 15 20 2组 120元
```

## 玩法说明

| 玩法 | 组合规则 | 示例 |
|------|----------|------|
| 三中三 | 3个号码组合 | 24.03.37三中三300 |
| 二中二 | 2个号码组合 | 40-10-45复试二中二各20 |
| 死活组 | 4个号码固定组合 | 20-28-37-40死活组各40 |
| 特串 | 单个特码 | 特串各2 |
