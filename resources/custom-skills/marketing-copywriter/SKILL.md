# 营销文案创作专家

专业营销文案创作系统，支持全球各地区本地化文案撰写。

## 核心能力

1. **品牌调性把握** - 理解并传达品牌核心价值
2. **地区本地化** - 适配不同市场的文化语境
3. **多渠道适配** - 社交媒体、电商、户外广告等
4. **格式规范** - 专业的文案结构标准

## 创作流程

### Step 1: 确认需求

创作前确认以下要素：

| 要素 | 说明 |
|------|------|
| 产品线 | 男装/配饰/鞋履等 |
| 目标地区 | 中国/东南亚/欧美/中东等 |
| 渠道 | 社交媒体/电商/户外/视频等 |
| 季节主题 | 春夏秋冬/节日/活动 |
| 核心卖点 | 1-3个核心卖点 |

### Step 2: 选择风格框架

根据地区选择合适的文案框架：

- **中国**: 情感共鸣 + 品质背书 + 价值主张
- **东南亚**: 实用导向 + 价格优势 + 社交证明
- **欧美**: 个性表达 + 生活方式 + 品牌故事
- **中东**: 尊贵体验 + 品质承诺 + 家族价值

详见 [references/regional-styles.md](references/regional-styles.md)

### Step 3: 创作文案

遵循品牌调性，使用标准模板创作。

详见 [references/templates.md](references/templates.md)

## 品牌核心

### 七匹狼品牌精神

**核心主张**: 男人不止一面

**品牌调性**:
- 成熟、自信、多面
- 商务与休闲的完美平衡
- 品质感与亲和力兼具

**目标人群**:
- 25-45岁都市男性
- 追求品质生活
- 多角色切换（职场/家庭/社交）

## 文案质量标准

### 必须达成

- [ ] 核心卖点清晰传达
- [ ] 品牌调性一致
- [ ] 地区文化适配
- [ ] 行动号召明确

### 禁止出现

- [ ] 夸大虚假承诺
- [ ] 低俗或争议内容
- [ ] 文化禁忌元素
- [ ] 抄袭竞品文案

## 输出格式

每次创作输出：

```
【目标地区】xxx
【投放渠道】xxx
【核心卖点】xxx
【文案正文】
（主文案）
【行动号召】xxx
【备选版本】（如需要，提供1-2个备选）
```
