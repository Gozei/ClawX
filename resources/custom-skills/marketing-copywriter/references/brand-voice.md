# 品牌调性指南

## 七匹狼品牌核心

### 品牌定位

**"男人不止一面"**

七匹狼致力于为现代都市男性提供全方位的着装解决方案，帮助他们在职场、社交、家庭等不同场景中展现独特的个人魅力。

### 品牌个性

| 维度 | 描述 | 文案体现 |
|------|------|----------|
| 成熟 | 沉稳、睿智、有担当 | "从容应对每个角色" |
| 自信 | 不张扬、有底气 | "内在力量，外在展现" |
| 多面 | 灵活切换、游刃有余 | "职场精英，生活达人" |
| 品质 | 讲究细节、追求卓越 | "每一针都是匠心" |

### 品牌故事要点

1. **奋斗者形象** - 从草根到精英的成长故事
2. **家庭责任** - 父亲、丈夫、儿子的角色担当
3. **职场进阶** - 专业、靠谱的商务形象
4. **生活品味** - 懂生活、有品质的休闲态度

### 文案调性规范

#### 语感特征

- **简洁有力** - 短句为主，节奏明快
- **真诚可信** - 不夸张，用事实说话
- **有温度** - 体现对用户生活的理解和关怀
- **有力量** - 传递自信和进取精神

#### 禁忌词汇

| 禁止使用 | 原因 | 替代方案 |
|----------|------|----------|
| 最、第一 | 广告法限制 | 卓越、领先、出色 |
| 必须、一定 | 过于强硬 | 值得、推荐、选择 |
| 奢华、顶级 | 与品牌定位不符 | 精致、品质、考究 |
| 年轻、潮流 | 目标人群不匹配 | 时尚、活力、现代 |

### 品牌金句库

**经典系列**:
- "男人不止一面，七匹狼男装"
- "穿出你的多面人生"
- "品质，是一种态度"

**场景系列**:
- "职场战场，从容应战"
- "回家，做回真实的自己"
- "周末时光，舒适开场"

### 视觉风格联动

文案需配合视觉风格：

- **色调**: 深蓝、灰色、米白为主
- **场景**: 商务会议、家庭聚会、户外休闲
- **人物**: 30-40岁成熟男性，自信从容
