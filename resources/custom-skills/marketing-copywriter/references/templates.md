# 文案模板库

## 一、社交媒体模板

### 小红书/Instagram 风格

```
【标题】emoji + 核心卖点 + 场景

【正文】
开头：痛点/场景引入
中间：产品解决方案（1-3个卖点）
结尾：行动号召

【标签】#品牌 #品类 #场景 #风格
```

### 示例

```
👔 春天来了，你的职场战袍准备好了吗？

三月会议季，一套得体的西装是男人的底气。
七匹狼春季商务系列，轻薄透气不闷汗，
从早会到晚宴，一穿到底不怯场。

🎯 春季新品上市，限时9折
📍 全国门店及官方旗舰店

#七匹狼 #春季男装 #职场穿搭 #商务西装
```

## 二、电商详情页模板

### 产品标题模板

```
【品牌】+ 【系列/风格】+ 【品类】+ 【核心卖点】+ 【适用场景】
```

示例：`七匹狼春季商务西装男 修身透气 纯色百搭 职场面试`

### 卖点描述模板

```
【产品亮点】
• 亮点1：具体描述 + 用户利益
• 亮点2：具体描述 + 用户利益
• 亮点3：具体描述 + 用户利益

【适用场景】
商务会议 | 日常通勤 | 重要面试 | 客户拜访

【尺码建议】
身高170-175cm，体重65-75kg，建议选择L码
```

## 三、朋友圈/微信群模板

### 短文案模板

```
【场景】+ 【痛点】+ 【解决方案】+ 【行动】

例：
春天忽冷忽热，穿什么成了难题？
七匹狼春季夹克，轻薄保暖两不误。
今天最后一天优惠，扫码下单 👇
```

### 长文案模板

```
【开篇】吸引注意力

【痛点】描述用户困扰

【方案】产品如何解决

【证明】用户评价/数据支撑

【行动】明确下一步
```

## 四、户外广告模板

### 公交站牌/地铁

```
主标题：6-10字核心主张
副标题：补充说明或卖点
品牌露出：Logo + 品牌名
行动号召：门店/网址
```

示例：
```
春天，从容开场
七匹狼春季新品
轻盈质感，自在出行
全国门店及官方商城
```

### 电梯广告

```
一句话文案 + 产品图 + 品牌Logo

例：
"职场C位，从得体着装开始"
```

## 五、视频脚本模板

### 短视频（15-30秒）

```
【0-3秒】开场钩子
- 提问/场景/悬念

【3-10秒】产品展示
- 核心卖点演示
- 细节特写

【10-15秒】情感升华
- 使用场景
- 用户感受

【15-30秒】行动号召
- 品牌露出
- 购买引导
```

### 示例脚本

```
[画面：清晨闹钟响起，男士起床穿衣]
旁白：每天早上，你有多少时间准备？

[画面：快速穿上七匹狼衬衫，整理领口]
旁白：七匹狼免烫衬衫，拿出来就能穿。

[画面：会议室、咖啡厅、接孩子场景快速切换]
旁白：从职场到生活，一套搞定。

[画面：品牌Logo，门店画面]
旁白：七匹狼春季新品，从容应对每一天。全国门店有售。
```
