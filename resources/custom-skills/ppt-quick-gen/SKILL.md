# PPT Quick Gen - 快速 PPT 生成

**核心目标**: 将 PPT 生成从"分钟级"优化到"秒级"

## 使用场景

1. **快速生成**: 用户说"做个PPT"，直接生成本地 PPTX 文件
2. **大纲转PPT**: 从已有大纲快速生成幻灯片
3. **模板复用**: 基于模板快速填充内容
4. **批量生成**: 多个 PPT 批量生成

## 工作流程

### 模式1: 快速生成（推荐）

```
用户请求 → 提取关键点 → 直接生成 PPTX → 返回文件
           (5秒)        (0.1秒)
```

**总耗时: ~5秒**

### 模式2: 标准生成

```
用户请求 → AI生成大纲 → 确认大纲 → 生成 PPTX → 返回文件
           (15秒)       (用户交互)  (0.1秒)
```

## 调用方式

### 直接调用脚本

```bash
python tools/ppt_quick_gen.py --title "标题" --slides "内容1|内容2|内容3" --output "output.pptx"
```

### JSON 输入

```bash
python tools/ppt_quick_gen.py --json '{
  "title": "产品介绍",
  "slides": [
    {"title": "封面", "content": "产品名称\n副标题"},
    {"title": "简介", "content": "产品定位\n核心功能"},
    {"title": "优势", "content": "优势1\n优势2\n优势3"}
  ]
}'
```

## 性能对比

| 方式 | 10页 PPT | 30页 PPT |
|------|---------|---------|
| 云端生成 | 30-60秒 | 60-120秒 |
| 本地快速生成 | **5秒** | **8秒** |

## 视觉模板

支持预设模板，快速切换风格：

1. **商务蓝** - 默认，专业商务风格
2. **科技绿** - 科技感，适合产品介绍
3. **简约白** - 极简风格，适合内部报告

```bash
python tools/ppt_quick_gen.py --template "tech" ...
```

## 与现有流程对比

| 方面 | 原流程 | 快速生成 |
|------|--------|---------|
| 内容生成 | AI 多轮对话 | AI 单次提取 |
| 确认环节 | 需要确认 | 可跳过 |
| 生成方式 | 外部工具/云端 | 本地 python-pptx |
| 总耗时 | 1-3分钟 | 5-10秒 |

## 注意事项

1. 快速生成适合"内容明确"的场景
2. 复杂设计需求仍需外部工具
3. 生成后可导出到 Canva/PPT 进一步美化
